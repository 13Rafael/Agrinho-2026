Estilização e Animações (style.css) Este arquivo aplica a paleta de cores solicitada (do verde escuro ao amarelo) usando gradientes, e adiciona um efeito de balanço suave nas plantas para dar a sensação de animação. CSS /* Reset básico de página */ * { margin: 0; padding: 0; box-sizing: border-box; font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; } body { background-color: #f7f9f6; color: #1b331e; /* Verde bem escuro para textos */ overflow-x: hidden; } /* Cabeçalho */ header { display: flex; justify-content: space-between; align-items: center; padding: 20px 8%; background-color: #0b2412; /* Verde Escuro */ color: #f4d03f; /* Amarelo */ } header .logo { font-size: 24px; font-weight: bold; color: #ffffff; } header .logo span { color: #f4d03f; } header nav a { color: #ffffff; text-decoration: none; margin-left: 20px; transition: color 0.3s; } header nav a:hover { color: #f4d03f; } /* Seção Hero (Principal) com Gradiente */ .hero { min-height: 85vh; display: flex; align-items: center; justify-content: space-between; padding: 0 8%; /* Gradiente do Verde Escuro passando pelo Verde Folha até o Amarelo Queimado */ background: linear-gradient(135deg, #0b2412 0%, #1e562c 50%, #d4ac0d 100%); color: #ffffff; position: relative; } .hero-content { max-width: 500px; z-index: 2; } .hero-content h1 { font-size: 48px; margin-bottom: 20px; color: #f4d03f; } .hero-content p { font-size: 18px; line-height: 1.6; margin-bottom: 30px; color: #e2ecc8; } .btn-main { background-color: #f4d03f; color: #0b2412; border: none; padding: 12px 28px; font-size: 16px; font-weight: bold; border-radius: 25px; cursor: pointer; transition: transform 0.2s, background-color 0.3s; } .btn-main:hover { background-color: #f1c40f; transform: scale(1.05); } /* Contêiner de Plantas Animadas */ .plant-container { display: flex; gap: 30px; align-items: flex-end; font-size: 80px; /* Tamanho dos emojis/desenhos */ position: relative; padding-bottom: 20px; z-index: 2; } /* Classe base para a animação das plantas */ .plant { display: inline-block; transform-origin: bottom center; animation: sway 3s ease-in-out infinite alternate; } /* Variações de tempo para parecer mais natural */ .leaf-1 { animation-duration: 2.5s; } .sprout { font-size: 100px; /* Destaque para o broto central */ animation-duration: 3.5s; } .leaf-2 { animation-duration: 3s; } /* Animação que faz as plantas balançarem como se houvesse vento */ @keyframes sway { 0% { transform: rotate(-5deg); } 100% { transform: rotate(5deg); } } /* Rodapé */ footer { background-color: #0b2412; color: #7a9a7d; text-align: center; padding: 20px; font-size: 14px; } /* Responsividade para celulares */ @media (max-width: 768px) { .hero { flex-direction: column; 🎨 2. Estilização e Animações (style.css) Este arquivo define a identidade visual. O fundo utiliza um gradiente que vai do verde escuro ao amarelo, e a propriedade @keyframes cuida da animação orgânica das plantas. CSS /* Reset de Margens e Box Sizing */ * { margin: 0; padding: 0; box-sizing: border-box; font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; } body { background-color: #f4f7f5; color: #0b2412; overflow-x: hidden; } /* Configuração do Menu/Header */ header { display: flex; justify-content: space-between; align-items: center; padding: 20px 8%; background-color: #0b2412; /* Verde Escuro */ } header .logo { font-size: 24px; font-weight: bold; color: #ffffff; } header .logo span { color: #f4d03f; /* Destaque Amarelo no ano */ } header nav a { color: #ffffff; text-decoration: none; margin-left: 24px; transition: color 0.3s ease; } header nav a:hover { color: #f4d03f; } /* Seção Hero com o Gradiente Solicitado */ .hero { min-height: 85vh; display: flex; align-items: center; justify-content: space-between; padding: 0 8%; /* Transição suave do Verde Escuro ao Amarelo */ background: linear-gradient(135deg, #0b2412 0%, #1e562c 45%, #f4d03f 100%); color: #ffffff; position: relative; } .hero-content { max-width: 550px; z-index: 2; } .hero-content h1 { font-size: 52px; margin-bottom: 15px; color: #f4d03f; text-shadow: 1px 2px 4px rgba(0, 0, 0, 0.3); } .hero-content p { font-size: 18px; line-height: 1.6; margin-bottom: 30px; color: #e2ecc8; } /* Botão de Ação */ .btn-main { background-color: #f4d03f; color: #0b2412; border: none; padding: 14px 32px; font-size: 16px; font-weight: bold; border-radius: 30px; cursor: pointer; transition: all 0.3s ease; box-shadow: 0 4px 15px rgba(0, 0, 0, 0.2); } .btn-main:hover { background-color: #ffffff; transform: translateY(-3px); } /* Elementos Visuais das Plantas */ .plant-container { display: flex; align-items: flex-end; gap: 20px; font-size: 90px; /* Tamanho dos elementos visuais */ z-index: 2; padding-bottom: 20px; } /* Base da Animação (Balanço Orgânico) */ .plant { display: inline-block; transform-origin: bottom center; /* Fixa a base para o balanço ser no topo */ animation: sway 3.5s ease-in-out infinite alternate; } /* Ajustes finos de tempo para dar naturalidade (não balançarem juntas) */ .leaf-left { animation-duration: 2.8s; } .sprout-center { font-size: 120px; /* Destaque para o broto principal */ animation-duration: 4s; } .leaf-right { animation-duration: 3.2s; } /* Regra da Animação de Balanço */ @keyframes sway { 0% { transform: rotate(-6deg); } 100% { transform: rotate(6deg); } } /* Rodapé */ footer { background-color: #0b2412; color: #8fa893; text-align: center; padding: 25px; font-size: 14px; } /* Responsividade para Dispositivos Móveis */ @media (max-width: 768px) { .hero { flex-direction: column; justify-content: center; text-align: center; padding: 60px 5%; } .hero-content { margin-bottom: 40px; } .plant-container { font-size: 70px; } .sprout-center { font-size: 90px; } }
